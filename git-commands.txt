# Git Commands to Upload to GitHub
# Run these commands in Command Prompt in the project directory:

# 1. Initialize Git repository
git init

# 2. Add all files
git add .

# 3. Commit with message
git commit -m "Initial commit: Claw-on-Daytona CLI tool"

# 4. Connect to your GitHub repository
git remote add origin https://github.com/TapanTalukdar004/claw-on-daytona.git

# 5. Rename branch to main
git branch -M main

# 6. Push to GitHub
git push -u origin main

# If you get an error that the repository doesn't exist:
# 1. Go to https://github.com/TapanTalukdar004/claw-on-daytona
# 2. Create an EMPTY repository (no README, .gitignore, or LICENSE)
# 3. Run the commands again starting from step 4

# After successful upload, install dependencies:
npm install

# Then test the CLI:
npm link
claw-on-daytona
